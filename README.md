## Основные проекты

### AI Call Analyzer

Локальный сервис анализа транскрипций звонков: классификация разговора, оценка применимых критериев, структурированный отчёт, задачи и review-сценарий для руководителя. Текстовый demo-режим работает без внешнего API; OpenAI, Google Sheets и MANGO представлены как опциональные, но не проверенные live-интеграции.

Стек: Python, FastAPI, Pydantic, SQLite, HTML/JavaScript, pytest.

[Репозиторий AI Call Analyzer](https://github.com/romabroha/ai-call-analyzer)

### Sales Assistant

Рекомендательная система для отдела продаж на основе истории заказов и товарного каталога. Реализованы сопутствующие товары, аналоги, топ клиента и режим новых товаров, объяснения рекомендаций, учёт остатков, API и локальный интерфейс.

Публичная версия использует только синтетические данные. Обмен с 1С описывается как подготовленный файловый/HTTP-контур без подтверждённого live-подключения.

Стек: Python, SQLite, REST-подобный JSON API, HTML/JavaScript, unittest.

[Репозиторий Sales Assistant](https://github.com/romabroha/sales-assistant)
